# Jargons are explained in this doc

**Service Catalog**

- Without SC Every time you create an invoice, you manually type the items.
- You first save your services once in the catalog
- Now when creating an invoice: you can just select for the catalog created

**Invitation**

```
Owner
  ↓
Creates Invitation
  ↓
Invitation (Pending)
  ↓
User signs up / logs in
  ↓
Accept Invitation
  ↓
Membership created
  ↓
User joins organization
```
